Description:

DevOps CI/CD project is a comprehensive demonstration of continuous integration and continuous deployment (CI/CD) practices implemented using Jenkins, Docker, Maven, and ReactJS. It showcases an automated pipeline with multiple stages, including building, staging/testing, publishing, deploying, and operating microservices-based applications.


Pipeline Stages

  • Building: Builds Docker images of a ReactJS application.
  • Staging & Testing: Executes Maven-built Selenium tests on a testing Dockerized environment to ensure application quality.
  • Publishing Latest Build: Publishes the latest build on Docker Hub with a designated build tag.
  • Deploying & Operating: Deploys Dockerized applications to a production-ready environment.

Repository Contents

  • Node-ReactJS Project: Contains the ReactJS application, runnable with commands for installation and execution.
  • Jenkinsfile: Defines the pipeline script, orchestrating the sequential stages of the CI/CD process.
  • Dockerfile: Specifies commands to assemble Docker images with dependencies for testing and deployment.
  • docker-compose.yml: Configures multiple Docker containers for deployment, combining and configuring them.
  • Maven App: Utilized for testing stages, integrating necessary dependencies.

This project exemplifies expertise in CI/CD methodologies, microservices architecture, containerization with Docker, and automation with Jenkins pipelines. It serves as a testament to proficiency in DevOps practices and showcases the ability to streamline software delivery pipelines for efficient development, testing, and deployment of microservices-based applications.
