Description:

The Ecommerce Market Application is a versatile Flutter-based project designed to facilitate seamless transactions between multiple users and sellers within a single platform. Leveraging Firebase Realtime Database as the backend, this application offers robust functionality across various modules, including authentication, inventory management, product browsing, cart management, and order tracking.

Key Modules

  1. Authentication (Login/Registration)

    Provides secure authentication functionalities allowing users to register for new accounts and login securely. Implements Firebase authentication mechanisms for enhanced security and ease of use.

  2. Admin Panel (Managing Product Inventory)

    Empowers administrators with a comprehensive panel to manage product inventory efficiently. Includes features for adding, updating, and removing products from the database.

  3. Main Page (Filter for Products and Favorite Picks)

    Offers a user-friendly interface showcasing the main page of the application. Includes advanced filtering options for users to easily browse through the available products. Incorporates a section for users to mark their favorite products for quick access.

  4. Product Page (Product Quantity)

    Displays detailed information about individual products, including descriptions, images, and prices. Allows users to select the desired quantity of products for purchase, enhancing customization and flexibility.

  5. Cart (Manage Products in Cart)

    Facilitates the management of products added to the shopping cart. Enables users to add, remove, and update quantities of products in the cart before proceeding to checkout.

  6. Orders (Previously Ordered Products)

    Provides users with a comprehensive overview of their previous orders. Allows users to track the status of their orders and view order history for reference.

This project showcases proficiency in mobile application development using Flutter, integrating real-time database functionalities with Firebase, and implementing robust authentication mechanisms. It demonstrates the ability to create intuitive user interfaces and efficient backend management systems.
