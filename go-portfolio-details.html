Description:

The Secure Login Authentication Project is a comprehensive implementation of secure authentication using JSON Web Tokens (JWT) and REST APIs. It incorporates encryption techniques to ensure data integrity and confidentiality throughout the authentication process. The project is structured with separate folders for authentication using Gin and Ionic for REST APIs in the Auth_jwt folder, and GraphQL implementation using gqlgen in the GraphQL_system folder.

Key Features

  1. Encrypted JWT for Login

    Utilizes JWT for user authentication, providing a secure mechanism for user login. Implements encryption techniques to safeguard sensitive user data during the login process.

  2. Encrypted JWT for Refreshing Expired Tokens

    Includes functionality to refresh expired JWT tokens, ensuring continuous user sessions without compromising security. Encryption methods are applied to maintain the confidentiality and integrity of refreshed tokens.

  3. Basic Request Testing

    Offers basic request testing functionalities to validate authentication after successful login. Utilizes Gin and Ionic for REST API testing in the Auth_jwt folder. Utilizes gqlgen for GraphQL API testing in the GraphQL_system folder.

  4. Postman Collection for Reference

    Provides a comprehensive Postman collection (gin.postman_collection.json) containing REST API endpoints for testing and reference. Enables seamless integration with Postman software for convenient testing and validation of APIs.

This project showcases a robust implementation of secure login authentication, incorporating industry-standard encryption techniques and leveraging popular frameworks such as gin-gonic, and gqlgen. It demonstrates proficiency in backend development, security protocols, and API testing.
