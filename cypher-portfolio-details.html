Cypher: End-to-End Encrypted Messenger

  • Category: App
  • Skills: Flutter, Javascript, Node.js, Mongodb
  • Project date: April, 2022
  • Project URL: https://github.com/tarush-r/End-To-End-Ecrypted-Chat-App

Description:

Cypher is an advanced instant messaging application designed with a focus on security and privacy. Utilizing end-to-end encryption (E2EE), Cypher ensures that all user chats and media exchanges are encrypted from sender to recipient, thereby preventing any unauthorized access or data breaches. The application employs a sophisticated encryption scheme that guarantees the server never has access to users' passwords or private keys, ensuring the highest level of security for user data. Authored and published a paper in Inventive Systems and Control, vol 436. Springer, Singapore

Key Features

  • End-to-End Encryption: All messages are encrypted using a private-public key pair, ensuring that only the intended recipient can decrypt and read the messages.
  • Secure Registration and Login: During registration, passwords are hashed and encryption keys are derived, ensuring that the server never has access to the user's actual password or private key. Similarly, during login, the client securely retrieves and decrypts the user's private key.
  • Comprehensive Functionality: Cypher offers a range of features including encrypted chatting, scheduled messaging, video calls via Agora, media and location sharing, profile management, password reset, chat deletion, account deletion, and multi-device logout options.
